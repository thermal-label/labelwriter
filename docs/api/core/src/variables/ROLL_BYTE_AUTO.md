# Variable: ROLL\_BYTE\_AUTO

> `const` **ROLL\_BYTE\_AUTO**: `48` = `0x30`

Wire byte for `ESC q 0x30` — automatic roll selection on the Twin
Turbo. The firmware picks an available roll. See LW 450 Series Tech
Ref p.16.
